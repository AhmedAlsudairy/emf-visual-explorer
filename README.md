# ⚡ EMF Visual Explorer

An interactive educational app that visually explains electromagnetic fields (EMF) around power transmission lines — no math needed.

## Features

- **11 Interactive Chapters** covering EMF fundamentals, 3-phase power, electric & magnetic fields, temperature, wind, dust, humidity, safety zones, and load/voltage effects
- **Live Animated Visuals** — canvas-based heatmaps, tower diagrams, wave animations, and particle simulations
- **Adjustable Parameters** — voltage (11–500 kV), load current, temperature, wind speed, humidity, distance
- **Safety Zone Calculator** with ICNIRP guideline reference
- **Mobile-Friendly** responsive design with dark theme

## Tech Stack

- **React 18** (CDN)
- **Babel Standalone** (in-browser JSX compilation)
- **Pure HTML/CSS/JS** — no build step required

## Deploy

### Vercel (recommended)
1. Push to GitHub
2. Import the repo on [vercel.com](https://vercel.com)
3. Deploy — no config needed, it auto-detects the static site

### Local
```bash
npx serve .
```
